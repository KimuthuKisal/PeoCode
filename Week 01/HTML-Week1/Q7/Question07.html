Using HTML5 Elements

The Navigation Section element

Represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents.

  • Home
  • About Us
  • Features
  • Pricing
  • Contact us

The Content Division element

This is a generic container for flow content. It has no effect on the content or layout until styled in some way using CSS

The Article Contents element

Represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable

Examples include: a forum post, a magazine or newspaper article, or a blog entry, a product card, a user-submitted comment, an interactive widget or gadget, or any other independent item of content.

The Generic Section element

My Section 1

This element represents a generic standalone section of a document, which doesn't have a more specific semantic element to represent it. Sections should always have a heading, with very few exceptions.

Footer

Represents a footer for its nearest sectioning content or sectioning root element. Typically contains information about the author of the section, copyright data or links to related documents.
